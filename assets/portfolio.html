    BPSC Persona is a project I created for the BP Student Council (BPSC) to help the student council members get a better understanding of their leadership personality, and be able to find people who are like minded. Using a similar system as MBTI to determine the personality, the user is brought through an adventure where their abilities will be tested. A personality card will then be generated with their name on it, which can be shared through a click of a button. This project was created using HTML, CSS, and JavaScript.
